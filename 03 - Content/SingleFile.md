# [[SingleFile]]  
___

## Description:
This tool allows for the cloning of single pages into completely self sufficient HTML pages

## Installation
`docker pull capsulecode/singlefile`

## Commands
`docker run capsulecode/singlefile "https://targetsite.com" > cloned-page.html`
