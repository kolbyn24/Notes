## Description:  

### running git locally
If you have a directory with a .git folder you can run git commands locally

```
git add hello_world.ps1
git commit
git log
git diff 4aefc023afb818866bd8c0920d438b44e76f642b a154424d8711ee481cf2e09b8acc6bbc844cddbe > patch.diff
```
